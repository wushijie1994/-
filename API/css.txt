1、背景
    background-attachment

2、边框
    outline：和border一样不过他在border的外面，这样一个容器就可以存在两个边框做样式
    border-spacing 属性设置相邻单元格的边框间的距离
    border-collapse 属性设置表格的边框是否被合并为一个单一的边框

3、列表属性
    list-style
    list-style-image
    list-style-position
    list-style-type

4、定位属性
    bottom，left，top，right
    display，float，position
    clear，overflow，z-index，visibility，
    clip：裁剪一张图像，唯一合法的形状值是：rect (top, right, bottom, left)

5、分页属性
    page-break-after
    page-break-before
    page-break-inside

6、表格
    border-collapse
    border-spacing
    caption-side设置表格标题的位置。
    empty-cells:设置是否显示表格中的空单元格（仅用于"分离边框"模式）
    table-layout:设置表格布局算法

7、文本
   direction:指定文本方向/书写方向,(ltr,rtl)
   text-transform:控制文本的大小写
       none	        默认。定义带有小写字母和大写字母的标准的文本。
       capitalize	文本中的每个单词以大写字母开头。
       uppercase	定义仅有大写字母。
       lowercase    定义无大写字母，仅有小写字母。
   word-wrap: normal|break-word;允许长的内容可以自动换行
